Experiment 005: Land-use Evidence Summary

Detailed implementation and result audit for using compact no-place land-use evidence summaries as classifier input for Qwen and Gemma.

1) Executive summary

Evidence text experiment

Experiment 005 tests whether a compact landuse_evidence_summary can replace generic summaries or raw article content as the classifier text source. The summaries are generated once from raw articles, stripped of place names, then classified by Qwen and Gemma with aligned and shuffled controls.

Main result: the land-use summaries contain real signal because aligned runs beat shuffled controls, but they do not replace raw content or the prior generic summaries. The extraction is too lossy for this classifier input.

4) Implementation details

Evidence summarization

  • Generator: LandUseEvidenceSummarizer.
  • Output schema includes summary, relevance, evidence types, no-place evidence sentences, and uncertainty.
  • Prompt version is LANDUSE_EVIDENCE_PROMPT_VERSION = 2.
  • Output is normalized for whitespace and rejected if it leaks private markers or place-title variants.
  • The evidence artifact records summary_no_place: true, seed 42, temperature 0.0.

Classification and evaluation

  • Text loading happens in classification/runner.py by reading landuse_evidence_summary from the evidence JSON.
  • Shuffled control uses landuse_evidence_summary_shuffled mapped to the same base text source.
  • Non-spatial metrics are produced by summarize_classification_experiment.py.
  • Spatial metrics are produced by evaluate_predictions_with_spatial_confidence.py.
  • Comparison outputs are collected in landuse_evidence_comparison_v1.

5) What a land-use summary looks like

Experiment 005 is where landuse_evidence_summary is introduced as a classifier text source. It is a compact, no-place synthesis extracted from the raw article, paired with metadata used later in experiment 006.

Example summary

Le territoire est majoritairement agricole avec des pâturages et des cultures, et une part significative de forêts. Des zones humides, notamment des tourbières, sont présentes et certaines sont protégées. Le relief est montagneux, situé à haute altitude.

Attached metadata

  • landcover_relevance: high
  • uncertainty: low
  • evidence_types: agriculture, forest, pasture, wetland, relief_or_geology
  • source evidence: acreage shares for agriculture, forest, pasture, wetland, and altitude/relief cues

Example from data/wiki/article_landuse_evidence_summaries.json, pageid 188561. The page title is deliberately not part of the summary text.

12) Interpretation and next step

  • Signal survives extraction: aligned evidence summaries beat shuffled controls for both models and tasks.
  • Extraction is lossy: the compact no-place summaries underperform raw content and generic summaries on primary metrics.
  • Spatial filtering remains useful: the 250m high-confidence subset improves results even for lossy evidence summaries.
  • Model split changes by task: Gemma is better on CORINE primary score, while Qwen is better on OSM and some minority CORINE classes.
  • Better use: keep land-use evidence as metadata, filtering, highlighting, or retrieval scaffolding rather than as a standalone classifier input.

The natural follow-up is not to replace content with summaries, but to combine evidence extraction with raw text: evidence cards, highlighted content, or retrieved evidence windows can preserve context while steering the classifier toward land-cover cues.
