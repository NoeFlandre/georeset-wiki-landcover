Experiment 004: Gemma 4 Rerun and Qwen Comparison

Detailed implementation and result audit for Gemma 4 31B IT Q4_0 on the frozen article-text classification protocol, plus direct comparison against the earlier Qwen baseline.

1) Executive summary

Model-only rerun

Experiment 004 changes the classifier model while keeping the same tasks, labels, prompts, text sources, shuffled-control logic, seed, temperature, and spatial-confidence table used in experiments 001 and 003. The rerun checks whether the article-text land-cover signal is robust to a second local LLM family.

Core conclusion: both Qwen and Gemma extract article-specific land-cover signal, but they fail differently. Gemma improves raw CORINE accuracy and OSM exact/micro metrics, while Qwen is stronger on balanced CORINE metrics and several minority classes.

10) Interpretation

  • Robust result: aligned-minus-shuffled deltas stay positive with Gemma, so the text signal from experiment 001 is not Qwen-specific.
  • Spatial result: Gemma also improves on spatially reliable subsets, matching the core conclusion from experiment 003.
  • Metric caveat: Gemma's higher raw accuracy is partly driven by better forest performance and stronger common-class alignment.
  • Balanced evaluation still matters: Qwen remains stronger on CORINE balanced accuracy and macro F1 in the important content comparisons.
  • OSM remains strict: exact match is harsh for multilabel sets, so micro F1, macro F1, and Jaccard should be read alongside it.

Best research phrasing: article text contains real land-cover signal across two local LLM families, but measured benefit depends strongly on metric choice, class imbalance, spatial ground-truth reliability, and class semantics.
